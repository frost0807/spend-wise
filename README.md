# spend-wise

### 프로젝트 개요

은행 거래내역 Api를 사용해 몇 번의 클릭으로 한 눈에 볼 수 있는 가계부를 볼 수 있는 서비스

### 기술스택

- Backend : Java, Springboot, SpringDataJpa, QueryDsl, SpringSecurity
- Frontend : Vue.js
- DB : MySql

### 아키텍쳐

### 주요기능

- 회원가입, 회원탈퇴, 로그인, 로그아웃
- 월별, 주별 지출내역 보여주기
- 지출 내역을 바탕으로 향후 소비에 대한 가이드 제공
- 사용자들의 지출 통계 정보 제공
